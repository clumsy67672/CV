Tentang Saya

Nama saya Azriel dan saat ini sedang menjalani pendidikan sarjana di bidang Teknik Elektro di Universitas Hang Tuah Surabaya. Sejak masa kuliah, saya menunjukkan minat dan bakat yang kuat dalam bidang teknik elektro, terbukti dengan prestasi yang saya raih selama ini. IPK saya selama ini adalah 3.83 yang menunjukkan komitmen dan dedikasi saya dalam belajar.

Selain itu, saya juga memiliki pengalaman dan keterampilan dalam berbagai hal. Saya mampu memprogram dengan baik menggunakan Arduino, Proteus, dan juga Adobe Illustrator. Saya juga memiliki keahlian dalam mengoperasikan kamera profesional dan mampu mengolah data menggunakan Excel dan Word. Berkat minat dan bakat yang saya miliki, saya juga aktif dalam berorganisasi. Keterlibatan saya dalam berbagai kegiatan organisasi membantu saya untuk meningkatkan soft skill dan network. Sebagai mahasiswa teknik elektro, saya memiliki kemampuan berbahasa Inggris yang baik dan mampu berkomunikasi dengan baik. Kemampuan ini sangat penting bagi saya karena memungkinkan saya untuk berkolaborasi dan bekerja sama dengan rekan-rekan dari berbagai negara.

Untuk itu, saya mencari kesempatan magang atau studi independen untuk meningkatkan keterampilan dan pengalaman. Saya yakin bahwa dengan mengambil bagian dalam proyek atau program magang, saya akan mampu mengasah kemampuan dan meningkatkan skill saya sebagai seorang teknisi elektro. Saya percaya bahwa dengan minat, bakat, dan keterampilan yang saya miliki, saya dapat memberikan kontribusi yang signifikan bagi perusahaan atau organisasi tempat saya bekerja. Saya siap untuk bekerja keras dan berkembang bersama perusahaan untuk mencapai tujuan bersama.
